## 웹 그리드시스템

셋째날 우리 수업에서 진행할 내용은 **웹 환경에 적합한 그리드 시스템 방법론**에 대해 알아보고, **실습**합니다.

* 본문(Body Text) 디자인(설계)
* 본문을 기반으로 하는 제목(Headings) 디자인
* 하이퍼링크(Hyperlinks) 디자인
* 수평/수직 리듬(Rhythm)
* 그리드 시스템 그리기
* 시스템에 맞춰 오브젝트 배치(레이아웃)
